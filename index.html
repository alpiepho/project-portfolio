Portfolio

These are some of the projects I have worked on recently. I expect this list to show my progression of interests as I learn what it means to be a Full Stack Web Developer.

Jan 2017: A Ruby based application to automate gathering baseball game data from an online service for youth/prep baseball teams. This tool uses the Watir Ruby Gem, an implementation of the Selenium API for automated web site testing. This is used to navigate to all the pages and scrap data from each page.

Jan 2017: This is my implementation of the full sample_app from the 'Ruby on Rails Tutorial' by Michael Hartl. This is a wellknown online book for learning Ruby on Rails. The book is 700 pages long and works through designing a Rails application that is a mini version of Twitter, complete with multiple views and database store.

Feb 2017: This is Ruby on Rails application that mirrors an existing Wordpress site used to promote a high school baseball player to prospective college coaches. This site was designed to be a template for other players. The player data is read from a configuration file on the site and populated to the Rails files.

Mar 2017: This is the 3rd iteration of an application concept to keep a list of technical terms along with description, notes, and related links. This is loosely based on the Angular2 'Heroes' tutorial. It uses a nAngular2 front end, Node and Express for a simple server, and MongoDB on the backend for storing list. Also wrote tools to save and reset/populate the DB.

Apr 2017: A simple reverse-polish-notation hex calculator Chrome Extension. Allows quick access to do simple hex operations and see the decimal value. Written in HTML, JavaScript, and jQuery. Also includes a Ruby/Watir Selenium test framework to verify all operations in a browser.

May 2017: The first project for freeCodeCamp, Basic Front End Development, is a simple HTML with Bootstrap CSS to create a tribute. I chose to build a tribute to an old science fiction book I read...and with all the talk of machine-learning and A.I., this book rings more and true.

May 2017: An older portfolio website itself was created with angular-cli (Angular2) and deployed on Heroku. The Angular2 code is trivial. This site was actually cloned from a template site I built so I could quickly generate new Angular2 sites and easily deploy to Heroku.

May 2017: The second project for freeCodeCamp, Basic Front End Development, This portfolio page was migrated to GitHub and GH Pages.

(Click on the image to go to that site. Most of the sites above are hosted on the free service of Heroku. This service will disable or 'sleep' the host virtual machine. My experiments show that the first access web can take 15-30 seconds longer due to the sites dyno waking up.)
